JWT/session client for Rainy API v3 dashboard endpoints. JWT/session client for Rainy API v3 dashboard-style endpoints.

This module intentionally separates session/JWT operations from RainyClient (API-key flows) to keep trust boundaries clear and the default SDK surface smaller.
